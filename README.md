# CatchByeongi

## 개발환경

* React Native + Supabase

* JDK(Java) 설치 (https://adoptium.net/)

Android 빌드에 필요 → Temurin JDK 17 권장

* Android Studio 설치 (https://developer.android.com/studio)

### 개발세팅

1. node.js LTS 설치

```nvm install lts```

2. npm 11.6.4 설치

```npm install -g npm@11.6.4```

3. React Native CLI 프로젝트 생성 및 앱 실행

```

npx @react-native-community/cli init MyApp

cd MyApp

// 에뮬레이터 켠 상태에서

npx react-native run-android

```
